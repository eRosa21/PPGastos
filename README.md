💰 PPGastos - Rastreador de Finanças Pessoais
Este é um projeto desenvolvido para gerenciar gastos diários de forma simples e intuitiva. A aplicação permite o registro, visualização e controle financeiro através de uma interface moderna construída com o framework Flet.


🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias:

Python 3.x: Linguagem principal do projeto. 

Flet: Framework para construção de interfaces (GUI) baseadas em Flutter.

SQLite: Banco de dados leve para armazenamento local dos registros.

Git & GitHub: Versionamento de código e hospedagem.

✨ Funcionalidades (em desenvolvimento)

[ ] Cadastro de novos gastos com descrição e valor.

[ ] Filtro de gastos por categoria.

[ ] Dashboard visual com resumo mensal.


🛠️ Como Executar o Projeto
Para rodar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:

git clone https://github.com/eRosa21/PPGastos.git

2. Instale as dependências:

(Certifique-se de ter o Python instalado)

pip install flet

3. Execute a aplicação:

python main.py
